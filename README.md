## 🍞 프로젝트 소개
```
  신뢰할 수 있는 정보로 더 나은 외식 경험을 제공합니다!

  최근 외식 선택의 어려움과 만족도 낮은 경험이 반복되고 있습니다. 
  이를 해결하기 위해 조건별 리뷰와 별점, 직접 등록 가능한 평가 시스템을 기반으로 한
  맛집 추천 플랫폼을 만들고자 합니다. 😊

```
